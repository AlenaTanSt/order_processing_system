## Запуск с параметрами по умолчанию:
```
.\build\solution\ops_app.exe
```

## Запуск с заданным количеством заказов:
```
.\build\solution\ops_app.exe 10000
```

## Запуск с нормальным завершением:
```
.\build\solution\ops_app.exe 10000 shutdown
```

## Запуск с принудительной отменой:
```
.\build\solution\ops_app.exe 10000 cancel
```

## Примечания

* В режиме shutdown CLI проверяет базовые инварианты (без прерывания выполнения).

* В режиме cancel допустимо, что часть заказов не будет доставлена.

* При малых значениях q_*_capacity и коротких push_timeout возможно появление неуспешных submit() из-за backpressure — это ожидаемое поведение
